Generic Fortran ( f77 ) This f77 generic version is recommended for most UNIX-like machines and is known to execute on Sun and RS6000 systems, and also DEC Alpha systems when swap.for is called first.
    Generic Fortran ( f90 ) This f90 generic version is recommended for most UNIX-like machines and is known to execute on Sun and RS6000 systems, and also DEC Alpha systems when swap.for is called first.
    Cray Fortran This version is set up specifically for use on a Cray system running UNICOS.
    DEC/VMS Fortran This version is set up specifically for use on a DEC/VMS system. It includes some macros.
    HP Fortran This version is set up specifically for use on an HP system.
    Microsoft Fortran This version is for use on PCs with Microsoft Fortran. It could probably be adapted for other PC Fortran compilers.
    RS6000 Fortran This version is set up specifically for use on an IBM RS6000 system.
    Sun Fortran This version is set up specifically for use on a Sun system.
